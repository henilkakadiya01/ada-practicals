# Repository Overview

This repository contains the laboratory implementations and practical exercises for the **Analysis and Design of Algorithms (ADA)** course.

The objective of this repository is to implement and analyze various **algorithmic techniques**, evaluate their **time and space complexity**, and understand their **applications in solving computational problems**.

The programs included demonstrate multiple algorithm design paradigms such as:

- Divide and Conquer
- Dynamic Programming
- Greedy Algorithms
- Graph Algorithms
- Backtracking
- Branch and Bound

---

# Course Outcomes

After completing the course, students will be able to:

1. Identify **time and space complexity** of algorithms.
2. Solve problems using **Divide and Conquer techniques**.
3. Analyze problems using **Greedy and Dynamic Programming approaches**.
4. Understand and solve **graph-based problems**.
5. Apply **Backtracking and Branch & Bound techniques** for complex problems.

These outcomes are defined in the GTU syllabus for ADA. :contentReference[oaicite:1]{index=1}

# Laboratory Experiments

1. Implement **Selection Sort, Bubble Sort, and Insertion Sort** and analyze execution time.
2. Implement **Linear Search and Binary Search**.
3. Implement **Merge Sort and Quick Sort**.
4. Solve **Greedy Problems**:
   - Making Change Problem
   - Activity Selection Problem
5. Implement **Greedy Algorithms**:
   - 0/1 Knapsack
   - Job Scheduling
6. Implement **Minimum Spanning Tree Algorithms**:
   - Prim’s Algorithm
   - Kruskal’s Algorithm
7. Solve problems using **Dynamic Programming**:
   - Making Change
   - Knapsack
8. Implement **Matrix Chain Multiplication**
9. Implement:
   - Longest Common Subsequence (LCS)
   - Shortest Common Supersequence (SCS)
10. Solve **4-Queen and 8-Queen Problems** using Backtracking
